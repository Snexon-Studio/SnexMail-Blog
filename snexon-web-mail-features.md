Dynamic mode is our intelligent selection mode. It automatically selects from our official domains and, if you're logged in and have added custom domains to your account, it will also include your personal domains in the random selection process.

**How It Works**:

**Domain Management Features**:
- Easy domain addition through settings
- DNS verification for domain ownership
- Unlimited domain storage per account
- Domain conflict resolution system

### Domain Conflict Resolution
When a user tries to add a domain that already exists in our database, our system implements an intelligent verification process:

1. **Conflict Detection**: System identifies existing domain
2. **Ownership Verification**: User receives a random TXT DNS record to add
3. **DNS Checking**: Our system verifies the DNS record
4. **Domain Assignment**: Domain is reassigned to new user with optional removal from previous account

This ensures fair domain usage while preventing conflicts.
